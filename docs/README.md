# US Stocks Autobot - GitHub Pages

This directory contains the generated HTML files for GitHub Pages.

Files are automatically generated by the main program and deployed via GitHub Actions.

## Files Structure

- `index.html` - Main page with historical picks list
- `YYYY-MM-DD.html` - Daily stock picks page
- `*.png` - Stock chart images
- `history.json` - Historical data (last 30 days)
